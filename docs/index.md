# Eliezer Go

Eliezer Go is an Android mobile app currently in early testing. The project explores location-based discovery, AR-style interactions, TON wallet connection flows, referrals, and real-device progress tracking.

The current focus is the Android app experience: installation, stability, wallet connection, app navigation, AR camera opening behavior, device trust checks, and feedback from real Android phones.
